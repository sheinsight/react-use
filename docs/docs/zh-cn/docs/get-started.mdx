## 🎨 使用 \{#usage}

下面是一个如何使用 [useAsyncFn](/reference/use-async-fn) Hook 的例子。访问 [Hooks 列表](/reference) 部分以查看所有可用的 Hooks。


```tsx
function App() {
  const { loading, error, run } = useAsyncFn(fetchData)

  return (
    <div>
      <div>{data}</div>
      <button onClick={run} disabled={loading}>
        {loading ? 'Loading...' : 'Fetch'}
      </button>
    </div>
  )
}
```

import { App } from '@/../../src/use-async-fn/demo'

<App />
