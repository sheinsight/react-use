# useLayoutEffectOnce

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

这是一个 React 钩子（Hook），其作用类似于 [useEffectOnce](/reference/use-effect-once)，但是内部使用了 [useIsomorphicLayoutEffect](/reference/use-isomorphic-layout-effect)。

## API \{#api}

```tsx
// 这是 `useIsomorphicLayoutEffect(callback, [])` 的别名，提供了更具语义的名称
useLayoutEffectOnce(callback)
```
