## API \{#api}

```tsx
import { create } from '@shined/reactive';
const [state, mutate] = useReactive(initialState, { create })
```

### 初始状态 InitialState \{#initial-state}

用于初始化状态的一个 `object` 对象。

### 选项 Options \{#options}

```tsx
export type UseReactiveOptions<State> = SnapshotOptions<State> & { create: typeof create }
```

[Reactive](https://sheinsight.github.io/reactive) 的选项，参见 [Reactive 创建选项](https://sheinsight.github.io/reactive/reference/root-api.html#create)。

### 返回值 \{#returns}

`state` 是当前状态，`mutate` 是一个对象，其改变将触发重渲染。
