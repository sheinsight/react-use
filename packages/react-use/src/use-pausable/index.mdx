## API

```tsx
const { isActive, pause, resume } = usePausable(initialStatus)
```

### InitialStatus

A `boolean` value of the initial internal `isActiveRef.current`, default to `false`.

### Returns

Returns contain [Pausable](/docs/features/pausable) instance that can be paused, resumed.

See [Pausable](/docs/features/pausable) for more details.
