## API \{#api}

它是从 `useKeyStroke` 演化而来的，有关更多详情，请查看 [useKeyStroke](/reference/use-key-stroke)。

```tsx
// 与 `useKeyStroke(keyFilter, handler, {...options, eventName: 'keypress'})` 相同
const stop = useKeyPress(keyFilter, handler, options)
```
