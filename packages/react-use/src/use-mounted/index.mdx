# useMounted

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

A React Hook that returns a [Ref Getter](/docs/features/ref-getter) that indicates whether the component is mounted or not.

## Demo

Try to click `Start Timer`, and then click the "Toggle mount" button in one second to see the result.

## API

```tsx
const isMounted = useMounted()
```

`isMounted` is a [Ref Getter](/docs/features/ref-getter) that returns a boolean value. 

See [Ref Getter](/docs/features/ref-getter) for more details.

### Returns

A Ref Getter that return a boolean value that indicates whether the component is mounted or not.
