# useResetState

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

一个类似于 [React.useState](https://react.dev/reference/react/useState) 的 React Hook，但额外提供了一个 `reset` 函数，用来将状态重置为初始值。

## 场景 Scenes \{#scenes}

- **状态初始化与重置场景：** 允许在组件生命周期内重置状态至初始值
- **交互元素状态管理：** 用于得分、步数等需要可重置的状态管理
- ...

## 用法 \{#usage}

```tsx
const [name, setName, resetName, initialName] = useResetState('张三')

// 修改状态
setName('李四')

// 重置状态为初始值
resetName()

// 重置状态为指定值，并同时修改初始值为指定值
resetName('王五')

console.log(name, initialName) // 王五, 王五
```
