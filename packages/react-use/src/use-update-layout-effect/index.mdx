# useUpdateLayoutEffect

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

A React Hook like [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect), but ignore the first invocation on mount, and use `useIsomorphicLayoutEffect` under the hood to support Server-side Rendering (SSR).
