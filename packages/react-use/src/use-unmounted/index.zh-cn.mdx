# useUnmounted

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

一个返回 [Ref Getter](/docs/features/ref-getter) 的 React Hook，用于判断组件是否已卸载。

## 演示 \{#demo}

尝试点击 `开始计时器`，然后在一秒内点击 "切换挂载" 按钮以查看结果。

## API \{#api}

```tsx
const isUnmounted = useUnmounted()
```

`isUnmounted` 是一个 [Ref Getter](/docs/features/ref-getter)，调用时返回一个布尔值。

前往 [Ref Getter](/docs/features/ref-getter) 了解更多。

### 返回值 \{#returns}

一个 Ref Getter，如果组件已卸载则返回 `true`。
