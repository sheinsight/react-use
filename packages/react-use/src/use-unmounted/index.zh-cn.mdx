# useUnmounted

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

一个返回 [Ref Getter](/docs/features/ref-getter) 的 React Hook，用于判断组件是否已卸载。

## 演示 \{#demo}

尝试点击 `开始计时器`，然后在一秒内点击 "切换挂载" 按钮以查看结果。
