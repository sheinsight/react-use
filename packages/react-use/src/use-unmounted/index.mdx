# useUnmounted

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

A React Hook that return a [Ref Getter](/docs/features/ref-getter) that returns `true` if the component is unmounted.

## Demo

Try to click `Start Timer`, and then click the "Toggle mount" button in one second to see the result.

## API

```tsx
const isUnmounted = useUnmounted()
```

`isUnmounted` is a [Ref Getter](/docs/features/ref-getter) that returns a boolean value. 

See [Ref Getter](/docs/features/ref-getter) for more details.

### Returns

A Ref Getter that returns `true` if the component is unmounted.
