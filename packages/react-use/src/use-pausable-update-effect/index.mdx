## API

```tsx
const pausable = usePausableUpdateEffect(callback, deps)
```

### Callback

Same as [React.useEffect](https://react.dev/reference/react/useEffect).

### Deps

Same as [React.useEffect](https://react.dev/reference/react/useEffect).

### Returns

Retuerns contain [Pausable](/docs/features/pausable) instance that can be paused, resumed.

See [Pausable](/docs/features/pausable) for more details.
