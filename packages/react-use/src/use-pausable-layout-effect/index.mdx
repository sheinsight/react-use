## API

```tsx
const pausable = usePausableLayoutEffect(callback, deps)
```

### Callback

Same as [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect).

### Deps

Same as [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect).

### Returns

Retuerns contain [Pausable](/docs/features/pausable) instance that can be paused, resumed.

See [Pausable](/docs/features/pausable) for more details.
