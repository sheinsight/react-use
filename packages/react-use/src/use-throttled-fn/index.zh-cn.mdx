## API

```tsx
const throttledFn = useThrottledFn(fn, { wait: 1000 })
```

### 目标函数 Fn \{#fn}

要节流的函数。

### 选项 Options \{#options}

```tsx
export type ThrottleOptions = {
  /**
   * 节流的时间间隔，单位为毫秒
   *
   * @defaultValue 默认值为 0
   */
  wait?: number
  /**
   * 是否在每个时间间隔的开始调用该函数
   *
   * @defaultValue 默认值为 true
   */
  leading?: boolean
  /**
   * 是否在每个时间间隔的结束调用该函数
   *
   * @defaultValue 默认值为 true
   */
  trailing?: boolean
}

export interface UseThrottledFnOptions extends ThrottleOptions {}
```

### 返回值 \{#returns}

返回节流后的函数，类型与输入的函数相同。
