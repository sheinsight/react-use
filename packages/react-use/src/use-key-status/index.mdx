## API

```tsx
const isPressed = useKeyStatus(keyFilter, handler, options)
```

### KeyFilter \{#key-filter}

See [KeyFilter](/reference/use-key-stroke/#key-filter) of `useKeyStroke` for more details.

### Handler

```tsx
export type UseKeyStatusHandler = UseKeyStrokeHandler;
```

See [Handler](/reference/use-key-stroke/#handler) of `useKeyStroke` for more details.

### Options

```tsx
export type UseKeyStatusOptions = Omit<UseKeyStrokeOptions, 'eventName' | 'dedupe'>;
```

See [Options](/reference/use-key-stroke/#options) of `useKeyStroke` for more details.

### Returns

A `boolean` value that indicates the press status of the key.
