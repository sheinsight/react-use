## API

```tsx
const { fps, ...controls } = useFps(options)
```

### Options

```tsx
export type UseFpsOptions = {
  /**
   * Calculate the FPS on every x frames.
   *
   * @defaultValue 10
   */
  every?: number
}
```

### Returns

Returns contain [Pausable](/docs/features/pausable) instance that can be paused, resumed.

See [Pausable](/docs/features/pausable) for more details.

```tsx
export type UseFpsReturns = Pausable & {
  /**
   * Current FPS
   */
  fps: number
}
```
