## API

Almost the same as [useManualStateHistory](/reference/use-manual-state-history), but with an additional [Pausable](/docs/features/pausable) instance return.

import { Tip } from '@/components'

<Tip type="pausable" />

```tsx
const { ...history, ...pausable } = useStateHistory(state, options)
```

### State

Same as [useManualStateHistory#state](/reference/use-manual-state-history/#state).

### Options

```tsx
export interface UseStateHistoryOptions<Raw, Serialized = Raw> extends UseManualStateHistoryOptions<Raw, Serialized> {
  deep?: boolean
}
```
