## API

It's derivated from `useKeyStroke`, see [useKeyStroke](/reference/use-key-stroke) for more details.

```tsx
// same as `useKeyStroke(keyFilter, handler, {...options, eventName: 'keyup'})`
const stop = useKeyUp(keyFilter, handler, options)
```
