# useUnmountedRef

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

A React Hook that helps to create a ref that can be checked if the component is unmounted. It is recommended to use [useUnmounted](/reference/use-unmounted) ([Ref Getter](/docs/features/ref-getter)) to get the unmount status

## Demo

Try to click `Start Timer`, and then click the "Toggle mount" button in one second to see the result.

### Returns

A React [Ref](https://react.dev/learn/referencing-values-with-refs) that can be checked if the component is unmounted.
