## API \{#api}

```tsx
const { isComposiing, data } = useInputComposition(elementTarget)
```

### 元素目标 ElementTarget \{#element-target}

`ElementTarget` 是一个联合类型，代表可以被定位的各种元素。

更多详情，请参见 [ElementTarget](/docs/features/element-target) 或 [ElementTarget 类型](/docs/features/element-target#element-target-types)。

### 返回值 \{#returns}

```tsx
interface InputCompositionState {
  /**
   * 输入框是否处于组合状态。
   */
  isComposing: boolean
  /**
   * 组合数据。
   */
  data: string
}

export interface UseInputCompositionReturns extends InputCompositionState {}
```
