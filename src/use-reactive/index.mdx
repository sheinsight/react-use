# useReactive

import { HooksType } from '@/components'

<HooksType category="ProUtilities" />

A React Hook that helps to use [Reactive](https://sheinsight.github.io/reactive) in React with ease.

:::tip
You need to install `@shined/reactive` to use this Hook.

```bash
# pnpm
pnpm install @shined/reactive

# npm
npm install @shined/reactive

# yarn
yarn add @shined/reactive
```

:::

## API

```tsx
const [state, mutate] = useReactive(initialState, options)
```

### InitialState

An `object` to initialize the state.

### Options

Options for [Reactive](https://sheinsight.github.io/reactive), see [Reactive Create Options](https://sheinsight.github.io/reactive/reference/api.html#root-create).

### Returns

`state` is the current state, `mutate` is an object whose changes will trigger a re-render.
