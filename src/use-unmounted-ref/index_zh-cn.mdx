# useUnmountedRef

import { HooksType } from '@/components'

<HooksType category="Lifecycle" />

使用该 React Hook 可以创建一个 ref，通过它可以检查组件是否已卸载。推荐使用 [useUnmounted](/reference/use-unmounted) ([Ref Getter](/docs/features/ref-getter)) 来获取组件卸载状态。

## 演示 Demo \{#demo}

尝试点击 `Start Timer`，然后在一秒内点击 “Toggle mount” 按钮，以查看结果。
