## API \{#api}

```tsx
const rafFn = useRafFn(fn, autoClean)
```

### 功能函数 Fn \{#fn}

需要在下一个帧上运行的 `function` 函数。

### 自动清理 AutoClean \{#auto-clean}

一个 `boolean` 布尔值，当组件卸载时将清理 raf。

### 返回值 Returns \{#returns}

一个 `function` 函数，将在下一个帧上运行 `fn` 函数。
