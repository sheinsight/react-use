# useRafFn

import { HooksType } from '@/components'

A React Hook that run a function only once on the next frame using [requestAnimationFrame](https://developer.mozilla.org/en-US/docs/Web/API/window/requestAnimationFrame).

## API

```tsx
const rafFn = useRafFn(fn, autoClean)
```

### Fn

A `funtion` that need to be run on the next frame.

### AutoClean

A `boolean` that will clean the raf when the component unmounts.

### Returns

A `function` that will run the `fn` on the next frame.
