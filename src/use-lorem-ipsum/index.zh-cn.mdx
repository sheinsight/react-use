## API \{#api}

```tsx
const text = useLoremIpsum(length) // 默认为 1 句
const text = useLoremIpsum(options)
```

### 句子长度 Length \{#length}

一个 `number` 类型的值，代表要生成的句子数量。

### 选项 Options \{#options}

```tsx
export interface UseLoremIpsumOptions {
  /**
   * 生成的文本句子的长度。
   *
   * @defaultValue 1
   */
  length?: number
  /**
   * 一个句子中的最小词语数量。
   *
   * @defaultValue ['.', '!', '?']
   */
  sentenceEnds?: string[]
  /**
   * 是否在渲染间保持同一个值。
   *
   * @defaultValue true
   */
  stable?: boolean
}
```

### 返回值 Returns \{#returns}

代表生成文本的一个 `string` 类型的值。
