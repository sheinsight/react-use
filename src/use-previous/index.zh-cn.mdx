## API

```tsx
const previous = usePrevious(state, isEqual)
```

### 状态 State \{#state}

你希望保持前一个值的状态。

### 相等性判断 IsEqual \{#is-equal}

一个 `function`，用于比较前一个值与当前值。默认值是 `Object.is`。

### 返回值 Returns \{#returns}

状态的前一个值。
