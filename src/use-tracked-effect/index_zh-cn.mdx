A React Hook for detecting dependencies that have **actually changed** between renders, only run in development mode.

## API

```tsx
useTrackedEffect(handler, deps)
```

### Handler

```tsx
export type TrackedEffectCallback = (changedDepIndexes: number[], previousDeps?: Deps, currentDeps?: Deps) => void
```

### Deps

Same as [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect).
