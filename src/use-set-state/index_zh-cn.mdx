## API

```tsx
const [state, setState] = useSetState(initialState, options)
```

### InitialState

Same as [React.useState](https://react.dev/reference/react/useState).

### Options

See [useSafeState#options](/reference/use-safe-state/#options) for more details.

### Returns

Same as [React.useState](https://react.dev/reference/react/useState).
