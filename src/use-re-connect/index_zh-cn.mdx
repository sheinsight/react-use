## API \{#api}

```tsx
useReConnect(fn, options)
```

### 函数 Fn \{#fn}

网络重新连接时将被调用的函数。

### 选项 Options \{#options}

```tsx
export interface UseReConnectOptions {
  /**
   * 注册重新连接事件监听器。
   */
  registerReConnect?: (callback: AnyFunc) => void
}
```
