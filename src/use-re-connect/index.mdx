## API

```tsx
useReConnect(fn, options)
```

### Fn

A function that will be called when the page is reconnect.

### Options

```tsx
export interface UseReConnectOptions {
  /**
   * Register reconnect event listener.
   */
  registerReConnect?: (callback: AnyFunc) => void
}
```
