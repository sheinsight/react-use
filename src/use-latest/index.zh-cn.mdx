`useLatest` 是一个 React 钩子（Hook），它返回一个指向最新值的 ref。它可以用来避免在定时器回调或异步上下文中出现 **陈旧状态**。

## API \{#api}

```tsx
const latestRef = useLatest(value)
```

### 值 Value \{#value}

你想要追踪的任何值。

### 返回值 Returns \{#returns}

一个 React ref (`React.MutableRefObject<T>`)
