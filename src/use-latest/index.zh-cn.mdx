# useLatest

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

一个返回一个指向最新值的 ref 的 React Hook，它可以用来避免在定时器回调或异步上下文中出现**过期闭包**。

## 演示 \{#demo}

import { App } from './demo'

<App />

## 用法 \{#usage}

```tsx
const latest = useLatest(count)
console.log(latest.current) // 最新的 count

const latest = useLatest({ callback, options })
console.log(latest.current.callback) // 最新的 callback
console.log(latest.current.options) // 最新的 options
```

## API \{#api}

```tsx
const latest = useLatest(value)
```

### 值 Value \{#value}

你想要追踪的任何值。

### 返回值 \{#returns}

一个 React ref (`React.MutableRefObject<T>`)
