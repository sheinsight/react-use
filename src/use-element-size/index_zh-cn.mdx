## API

```tsx
const { width, height, ...useResizeObserverReturns } = useElementSize(elementTarget, defaultValue, options)
```

### 元素目标 ElementTarget \{#element-target}

import { Tip } from '@/components'

<Tip type="target" />

### 默认值 DefaultValue \{#default-value}

```tsx
export type ElementSize = {
  /**
   * 元素的宽度。
   */
  width: number
  /**
   * 元素的高度。
   */
  height: number
}
```

### 选项 Options \{#options}

`useResizeObserver` 的选项，详情见 [useResizeObserver#options](/reference/use-resize-observer/#options)。

### 返回值 Returns \{#returns}

`useResizeObserver` 的返回值，详情见 [useResizeObserver#returns](/reference/use-resize-observer/#returns)。

```tsx
export type UseElementSizeReturns = ElementSize & UseResizeObserverReturns
```
