一个 React Hook，创建一个记忆化的值，只有在依赖项更改时才会重新执行创建。

## API

```tsx
const value = useCreation(factoryFn, deps)
```

### 工厂函数 FactoryFn \{#factory-fn}

返回要进行记忆化的值的函数。

### 依赖项 Deps \{#deps}

触发重新创建记忆化值的依赖项数组。
