## API

```tsx
const controls = useIntervalFn(callback, interval, options)
```

### Callback

A `function` that will be called every `interval` milliseconds.

### Interval

A `number` representing the interval in milliseconds.

### Options

```tsx
export interface UseIntervalFnOptions {
  /**
   * Whether to start the interval immediately on mounted
   *
   * @default true
   */
  immediate?: boolean
  /**
   * Whether to execute the callback immediately before the interval starts
   *
   * @default false
   */
  immediateCallback?: boolean
}
```
