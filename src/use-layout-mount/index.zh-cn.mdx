## API \{#api}

```tsx
useLayoutMount(callback, strictOnce)
```

### 回调 Callback \{#callback}

组件挂载时（useLayoutEffect）会被调用的一个 `function`，可以是 **异步** 的。

### 严格一次 StrictOnce \{#strict-once}

:::warning

`strictOnce` 选项 **不** 推荐使用，因为它 [损害](https://react.dev/learn/synchronizing-with-effects#dont-use-refs-to-prevent-effects-from-firing) 了 React 18 严格模式原有的初衷。

:::

一个 `boolean`，指示在 React 18 的 [严格模式](https://react.dev/reference/react/StrictMode) 中处理程序是否仅被调用一次。默认为 `false`。
