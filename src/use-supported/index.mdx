## API

```tsx
const isSupported = useSupported(callback, deps)
```

### Callback

A `function` that returns a boolean value to check if the feature is supported.

### Deps

An array of dependencies to re-run the check when the value changes.

### Returns

A `boolean` value that indicates if the feature is supported.
