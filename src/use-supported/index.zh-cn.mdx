## API

```tsx
const isSupported = useSupported(callback, deps)
```

### 回调函数 Callback \{#callback}

一个函数，其返回值会被转为布尔值用于检查是否支持该特性。

### 依赖 Deps \{#deps}

`React.DependencyList`, 一个在值改变时重新运行检查的依赖项数组。

### 返回值 Returns \{#returns}

一个指示特性是否受支持的 `boolean` 值。
