# useIsomorphicLayoutEffect

import { HooksType } from '@/components'

<HooksType {...frontmatter} />

A React Hook that will automatically use [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect) in client side and [React.useEffect](https://react.dev/reference/react/useEffect) in server side to avoid the warning of `useLayoutEffect` in Server-side Rendering (SSR).

Try to change count and then open DevTools to see the effect.
