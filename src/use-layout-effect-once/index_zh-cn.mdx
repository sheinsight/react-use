## API \{#api}

```tsx
// 这是 `useIsomorphicLayoutEffect(callback, [])` 的别名，提供了更具语义的名称
useLayoutEffectOnce(callback)
```

与 [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect) 相同。
