## API

```tsx
// alias to `useIsomorphicLayoutEffect(callback, [])` with more semantic name
useLayoutEffectOnce(callback)
```

### Callback

Same as [React.useLayoutEffect](https://react.dev/reference/react/useLayoutEffect).
