# useResetState

import { HooksType } from '@/components'

类似于 [React.useState](https://react.dev/reference/react/useState) 的一个 React 钩子（Hook），但提供了一个 `reset` 函数，用来将状态重置为初始值。

## API \{#api}

```tsx
const [state, setState, reset, initialState] = useResetState(initialState, options)
```

### 初始状态 InitialState \{#initial-state}

状态的初始值。

### 选项 Options \{#options}

有关更多详情，请参见 [useSafeState#options](/reference/use-safe-state/#options)。

### 返回值 Returns \{#returns}

```tsx
export function useResetState<T>(
  initialState: Gettable<T>,
  options?: UseResetStateOptions,
): [T, ReactSetState<T>, (initialState?: T) => void, T]
```
