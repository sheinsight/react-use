## API \{#api}

它是从 `useKeyStroke` 派生的，更多详情请见 [useKeyStroke](/reference/use-key-stroke)。

```tsx
// 与 `useKeyStroke(keyFilter, handler, {...options, eventName: 'keydown'})` 相同
const stop = useKeyDown(keyFilter, handler, options)
```
