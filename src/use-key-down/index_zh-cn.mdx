## API

import Link from '@docusaurus/Link'

It's derivated from `useKeyStroke`, see <Link to="/reference/use-key-stroke">`useKeyStroke`</Link> for more details.

```tsx
// same as `useKeyStroke(keyFilter, handler, {...options, eventName: 'keydown'})`
const stop = useKeyDown(keyFilter, handler, options)
```
