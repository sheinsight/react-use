## API

```tsx
const debouncedFn = useDebouncedFn(fn, options)
```

### Fn

The `function` to debounce.

### Options

```tsx
export type DebounceOptions = {
  /**
   * Time to wait before invoking the function
   *
   * @default 0
   */
  wait?: number
  /**
   * immediately invoke before the timeout
   *
   * @default false
   */
  leading?: boolean
  /**
   * invoke the function after the timeout
   *
   * @default true
   */
  trailing?: boolean
}

export type UseDebouncedFnOptions = DebounceOptions & {}
```

### Returns

The debounced `function`, same as the input function.
