## API \{#api}

```tsx
const { fps, ...controls } = useFps(options)
```

### 选项 Options \{#options}

```tsx
export type UseFpsOptions = {
  /**
   * 每 x 帧计算一次 FPS。
   *
   * @defaultValue 10
   */
  every?: number
}
```
