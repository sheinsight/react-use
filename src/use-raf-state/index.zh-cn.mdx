# useRafState

import { HooksType } from '@/components'

用于在下一帧使用 [requestAnimationFrame](https://developer.mozilla.org/en-US/docs/Web/API/window/requestAnimationFrame) 更新状态，以获得更好性能的 React Hook，类似于 [React.useState](https://react.dev/reference/react/useState)。

## API \{#api}

```tsx
const [state, setState] = useRafState(initialState, options)
```

### 初始状态 InitialState \{#initial-state}

与 [React.useState](https://react.dev/reference/react/useState) 相同。

### 选项 Options \{#options}

更多详情参见 [useSafeState#options](/reference/use-safe-state/#options)。

### 返回值 Returns \{#returns}

与 [React.useState](https://react.dev/reference/react/useState) 相同。
