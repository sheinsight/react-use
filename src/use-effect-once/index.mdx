# useEffectOnce

import { HooksType } from '@/components'

A React Hook like [React.useEffect](https://react.dev/reference/react/useEffect) but only run once, with more **semantic** name. It's merely an alias to `useEffect` with an empty dependency array.

## API

```tsx
// alias to `useEffect(callback, [])` with more semantic name
useEffectOnce(callback)
```

### Callback

Same as [React.useEffect](https://react.dev/reference/react/useEffect).
