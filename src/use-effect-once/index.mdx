# useEffectOnce

import { HooksType } from '@/components'

<HooksType category="Lifecycle" />

A React Hook like [React.useEffect](https://react.dev/reference/react/useEffect) but only run once, equivalent to `useEffect(callback, [])`, but with more **semantic** name. It's merely an alias to `useEffect` with an empty dependency array.

## API

```tsx
// alias to `useEffect(callback, [])` with more semantic name
useEffectOnce(callback)
```

### Callback

Same as [React.useEffect](https://react.dev/reference/react/useEffect).
