## API

It's derivated from `useKeyStroke`, see [useKeyStroke](/reference/use-key-stroke) for more details.

```tsx
// same as `useKeyStroke(keyFilter, handler, {...options, eventName: 'keypress'})`
const stop = useKeyPress(keyFilter, handler, options)
```
