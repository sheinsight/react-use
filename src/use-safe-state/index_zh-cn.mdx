一个类似于 [React.useState](https://react.dev/reference/react/useState) 的 React Hook，包括一个可选的深比较功能（默认为 `false`）。

有关 `useSafeState` 优势的详细解释，请参阅 [安全状态](/docs/optimization/safe-state) 文档。

## API

```tsx
const [state, setState] = useSafeState(initialState, options)
```

### 初始状态 InitialState \{#initial-state}

The initial state of the state.

### 选项 Options \{#options}

```tsx
export type UseSafeStateOptions = {
  /**
   * Deeply compare the new state with the old state before updating.
   * If true, only update the state when the new state is different from the old state.
   *
   * @defaultValue false
   */
  deep?: boolean
}
```

### 返回值 Returns \{#returns}

Same as [React.useState](https://react.dev/reference/react/useState).
