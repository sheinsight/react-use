# useSafeState

import Admonition from '@theme/Admonition'

一个类似于 [React.useState](https://react.dev/reference/react/useState) 的 React Hook，它包含一个可选的深度比较功能（默认值为 `false`）。

有关 `useSafeState` 优势的详细解释，请参阅 [安全状态](/docs/optimization/safe-state) 文档。

## API

```tsx
const [state, setState] = useSafeState(initialState, options)
```

### 初始状态 Initial-State \{#initial-state}

状态的初始值。

### 选项 Options \{#options}

```tsx
export type UseSafeStateOptions = {
  /**
   * 在更新前，深度比较新旧状态。
   * 如果为 true，只有当新状态与旧状态不同时，才更新状态。
   *
   * @defaultValue false
   */
  deep?: boolean
}
```

### 返回值 Returns \{#returns}

与 [React.useState](https://react.dev/reference/react/useState) 相同。
