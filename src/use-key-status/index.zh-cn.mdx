## API \{#api}

```tsx
const isPressed = useKeyStatus(keyFilter, handler, options)
```

### 键过滤器 KeyFilter \{#key-filter}


请参见 `useKeyStroke` 的 [KeyFilter](/reference/use-key-stroke/#key-filter) 了解更多详细信息。

### 处理函数 Handler \{#handler}

```tsx
export type UseKeyStatusHandler = UseKeyStrokeHandler;
```

请参见 `useKeyStroke` 的 [Handler](/reference/use-key-stroke/#handler) 了解更多详细信息。

### 选项 Options \{#options}

```tsx
export type UseKeyStatusOptions = Omit<UseKeyStrokeOptions, 'eventName' | 'dedupe'>;
```

请参见 `useKeyStroke` 的 [Options](/reference/use-key-stroke/#options) 了解更多详细信息。

### 返回值 Returns \{#returns}

一个 `boolean` 值，表示按键的按下状态。
