# useUnmounted

import { HooksType } from '@/components'

<HooksType category={frontmatter.category} />

A React Hook that return a [Ref Getter](/docs/features/ref-getter) that returns `true` if the component is unmounted.

## Demo

Try to click `Start Timer`, and then click the "Toggle mount" button in one second to see the result.
