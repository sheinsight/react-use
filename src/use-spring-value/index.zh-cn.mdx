## API \{#api}

```tsx
const spring = useSpringValue(start, end, options)
```

### 起始值 Start \{#start}

`number`, 弹簧的起始值。

### 终止值 End \{#end}

`number`, 弹簧的终止值。

### 选项 Options \{#options}

```tsx
export type UseSpringValueConfig = {
  /**
   * 弹簧的刚度。
   */
  stiffness?: number
  /**
   * 弹簧的阻尼。
   */
  damping?: number
  /**
   * 弹簧的质量。
   */
  mass?: number
  /**
   * 是否立即触发。
   */
  immediate?: boolean
  /**
   * 弹簧动画的精度。
   */
  precision?: number
}
```
