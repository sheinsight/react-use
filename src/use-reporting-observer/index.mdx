# useReportingObserver

import { HooksType } from '@/components'

A React Hook that helps to use [ReportingObserver API](https://developer.mozilla.org/en-US/docs/Web/API/ReportingObserver) to report errors and warnings with ease.

## API

```tsx
const { observerRef, isSupported, ...controls } = useReportingObserver(callback, options)
```

### Callback

`ReportingObserverCallback` function that will be called when the observer is triggered.

### Options

```tsx
export type UseWebObserverOptions = {
  /**
   * Start the observer immediate after calling this function
   *
   * @defaultValue true
   */
  immediate?: boolean
}

interface ReportingObserverOptions {
  buffered?: boolean
  types?: string[]
}

export interface UseReportingObserverOptions extends UseWebObserverOptions, ReportingObserverOptions {}
```
