## API \{#api}

```tsx
const { loadingSlow, loading, value, run, error } = useLoadingSlowFn(fn, options):
```

### 函数 Fn \{#fn}

一个返回 promise 的函数。

### 选项 Options \{#options}

```tsx
export interface UseLoadingSlowOptions extends UseAsyncFnOptions {
  /**
   * 确定加载是否慢的超时时长（毫秒）。
   * 
   * @defaultValue 3_000 ms
   */
  loadingTimeout?: number
}
```

### 返回值 Returns \{#returns}

```tsx
export interface UseLoadingSlowReturns<T extends AnyFunc> extends UseAsyncFnReturns<T> {
  /**
   * 是否加载慢。
   */
  loadingSlow: boolean
}
```

有关更多详情，请参见 [useAsyncFn#api](/reference/use-async-fn/#api)。
