## API \{#api}

它是从 `useKeyStroke` 派生的，有关更多详情，请参见 [useKeyStroke](/reference/use-key-stroke)。

```tsx
// 与 `useKeyStroke(keyFilter, handler, {...options, eventName: 'keyup'})` 相同
const stop = useKeyUp(keyFilter, handler, options)
```
